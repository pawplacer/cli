# Security Policy

## Supported Versions

Security fixes are applied to the latest published version of `pawplacer-cli`.

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Email security reports to:

```text
contact@pawplacer.com
```

Include:

- A description of the issue
- Steps to reproduce, if available
- Affected versions
- Any relevant logs, payloads, or proof of concept details

We will acknowledge valid reports as soon as practical and coordinate a fix before public disclosure.

## API Keys

Do not commit PawPlacer API keys, tokens, or credentials. Prefer environment variables or local configuration files excluded from version control.
